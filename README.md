# 社畜サバイバル

60秒で終わるクリック型ミニゲームです。誰から来た予定かを見ながらタスクを処理し、体力・信頼・集中力・炎上度・残タスクを管理します。

## 起動

```bash
pnpm install
pnpm dev
```

## ビルド

```bash
pnpm build
```

## デモ手順

1. `出社する` を押す。
2. 開始前に、社長・部長・先輩・同僚・後輩・自分の役職、名前、プリセット、画像を設定できる。
3. 降ってくるカードの人物、役職、タスク種別アイコンを見てクリックする。
4. 仕事を処理すると信頼は上がるが、体力と集中力が減る。人物によって追加補正が入る。
5. 社長タスクを見逃すと炎上度が大きく増える。
6. 効果が分からないときは、プレイ中の `効果一覧` を開く。開いている間はゲーム時間が止まります。
7. プレイ中や結果画面の `ホーム` / `ホームへ` から開始画面に戻れる。
8. 体力0または炎上度100で即終了。60秒耐えると称号とスコアが出る。

## 実装メモ

- バックエンド、認証、共有ランキングはありません。
- 最高記録と人物設定をブラウザの `localStorage` に保存します。
- 効果音はWeb Audio APIで生成しています。素材ファイルは使っていません。
- 初期状態では全員にデモ用のSVG人物画像が入っています。
- 画像をアップロードするとブラウザ内で縮小し、人物設定として保存します。画像を外した場合はCSSと文字アイコンのプリセットを表示します。
